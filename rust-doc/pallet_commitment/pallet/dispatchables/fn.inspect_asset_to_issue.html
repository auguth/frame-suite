pub fn inspect_asset_to_issue<T: Config<I>, I: 'static>()
Expand description

Queries the total amount of assets currently recorded as pending issuance.

This value reflects the amount tracked in AssetToIssue, representing assets that have been accounted for as “to be minted” but may not yet be reflected in the underlying asset system due to lazy execution.

The returned value is purely an accounting snapshot and does not guarantee that minting has already occurred.
