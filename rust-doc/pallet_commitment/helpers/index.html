Source
Expand description

Implementation of low-level commit-helpers traits for the internal CommitHelpers Type.

CommitHelpers implements traits:

  • CommitBalance
  • CommitDeposit
  • CommitWithdraw
  • CommitOps
  • CommitInspect
  • PoolOps
  • IndexOps

Local Tests for these traits are covered in tests .
